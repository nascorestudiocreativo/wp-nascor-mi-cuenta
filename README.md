# Nascor Mi Cuenta Personalizado 🚀

Un plugin para WordPress que transforma el aburrido panel de "Mi Cuenta" en una experiencia moderna, unificada y fluida gracias a su **navegación AJAX**. 

Lo mejor de este plugin es su **naturaleza dual**: se integra perfectamente con los endpoints de WooCommerce si está activo, pero también funciona como un panel de gestión de perfil nativo de WordPress si no utilizas WooCommerce.

## ✨ Características Principales

* **Navegación AJAX sin recargas:** Transiciones suaves entre las pestañas del panel.
* **Compatibilidad Dual:**
    * **Modo WooCommerce:** Reemplaza la navegación estándar de WooCommerce, manteniendo los endpoints originales (pedidos, descargas, direcciones, etc.) con una UI mejorada.
    * **Modo Nativo (Stand-alone):** Si WooCommerce no está instalado, crea un panel donde los usuarios pueden editar su perfil de WordPress (nombre, correo, contraseña).
* **Totalmente Personalizable:** Panel de opciones en el administrador de WordPress (`Ajustes > Mi Cuenta (Nascor)`) para modificar:
    * Color principal (Botones y enlaces activos)
    * Color de fondo general y del menú lateral (sidebar)
    * Color de texto
    * Radio de los bordes (Border radius)
* **Diseño Responsive:** Adaptado para dispositivos móviles, transformando las tablas de pedidos y descargas en tarjetas legibles.
* **Protección de contenido:** Muestra automáticamente un formulario de inicio de sesión si el usuario no está logueado.

## 📦 Instalación

1. Descarga el repositorio como un archivo `.zip`.
2. En tu panel de WordPress, ve a **Plugins > Añadir nuevo > Subir plugin**.
3. Selecciona el archivo `.zip` y haz clic en **Instalar ahora**.
4. Activa el plugin.

## 🛠️ Uso

Simplemente pega el siguiente shortcode en la página que desees usar como tu área de "Mi Cuenta":

`[nascor_mi_cuenta]`

## ⚙️ Configuración

Una vez activado, ve a **Mi Cuenta (Nascor)** en el menú lateral del administrador de WordPress. Allí podrás ver el estado de la integración (si detecta WooCommerce o no) y acceder al panel de colores con una **vista previa en vivo** para ajustar el diseño al branding de tu sitio web.
